<<<
=== drop (output)

[.cmd-header]
Drop output packets

This output plugin simply drops all packets.
This plugin is useful when the interesting work is done by the various packet processing plugins
and the actual output packets are useless.

[.usage]
Usage

[source,shell]
----
$ tsp -O drop [options]
----
